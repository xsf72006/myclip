myclip

A lightweight, native clipboard history for macOS. Press ⌘⇧C to pop up a small panel showing your recent copies (text and images), search through them, and pick one to paste. Single binary, no Electron, no background daemons other than the app itself.

Features

  • Global hotkey to toggle the panel (default ⌘⇧C, configurable in Settings)
  • Text and images — TIFF or PNG payloads are stored as PNG on disk
  • Search the history with a fuzzy substring match
  • Skips passwords generated by Apple Passwords, 1Password, Bitwarden, etc. by honouring the nspasteboard.org convention
  • Default visible: 10, max stored: 50 (oldest dropped automatically)
  • Menu-bar only — no Dock icon, no menu bar shown for the app itself
  • Auto-starts at login via a LaunchAgent
  • Cmd+C still works — myclip uses a different chord and doesn't intercept system copy

Requirements

  • macOS 14 (Sonoma) or later
  • Xcode Command Line Tools (xcode-select --install — full Xcode not required)

Quick Install

git clone https://github.com/<your-username>/myclip.git
cd myclip
make install

That builds the app, copies it to /Applications/myclip.app, and registers a LaunchAgent so it starts automatically every time you log in. The first launch puts a 📋 icon in your menu bar; press ⌘⇧C to pop up the panel.

Just Build It (don't install)

make build         # produces dist/myclip.app
make run           # build and open

Configure the Hotkey

Click the 📋 menu-bar icon → Settings…

You'll see the current shortcut and a row of presets (⌘⇧C, ⌘⇧V, ⌘⌥V, ⌃⇧Space). For anything else, type a spec like command+shift+k into the custom field and hit Apply. The recognised modifiers are command, shift, option, control.

Customising the Filter

The default already skips password-manager pastes. If you want to tighten or loosen the rules — e.g. drop pure-whitespace text, cap image size, block a specific source app — you only edit one file:

Sources/myclip/Pasteboard+Filters.swift

The shouldAccept(types:text:image:) function is the single decision point. It's intentionally short (≈5 lines) so you can read it end-to-end and add a clause without touching anything else, then make install to roll out.

Where Data Lives

~/Library/Application Support/myclip/
    history.json        ← metadata (50 most recent items)
    img-*.png           ← image payloads

Clear history in the menu bar wipes both. Uninstalling does not delete this folder so you don't lose history if you reinstall; remove it manually if you want a clean slate.

Uninstall

make uninstall

Removes /Applications/myclip.app and the LaunchAgent. History folder is left in place (see above).

Troubleshooting

The hotkey doesn't fire.

Another app may already own the same shortcut (some terminals grab ⌘⇧C). Change it under Settings → presets, or pick a custom combo.

My passwords show up in history.

The skip rule relies on the source app tagging the pasteboard with one of org.nspasteboard.{Concealed,Transient,AutoGenerated}Type. Apple Passwords, 1Password, and Bitwarden's desktop app all do; some browser extensions don't. If a specific app misbehaves, you can add an app-specific block in Sources/myclip/Pasteboard+Filters.swift.

Logs.

LaunchAgent stdout/stderr go to /tmp/myclip.out.log and /tmp/myclip.err.log.
